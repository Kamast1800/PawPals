# PawPals
Helping pet owners find safe, compatible doggy playdates.

## Overview
PawPals is a mobile-first web app that connects dog owners to find safe, compatible playmates for their pets. Users can create detailed dog profiles, browse nearby matches, and coordinate playdates.

## Tech Stack
- **Client**: React PWA (Vite)
- **API**: Express (Node.js)
- **Database**: Supabase (Postgres + Auth + Storage)

## Repository Structure
